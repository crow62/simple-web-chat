Стартовая страница чата по адресу: localhost:8080/simple-web-chat

Данный чат имеет следующие возможности:

1) Можно залогиниться и войти в чат, при этом если такое имя уже используется, выводится сообщение;
Также осуществляется проверка на ввод в поле пустого значения.(RegisteredUsersFilter + required в форме)
2) Войти на страницу чата нельзя, если еще не залогинился (chat.jsp), правда можно перейти по адресу
сервлета (/chat),с этим ничего не смог сделать. (ViewChatFilter)
3) Список вошедших пользователей и отправленные ими сообщения для всех пользователей
отображаются запросами ajax по таймауту (messages.js, users.js).
4) Имеется возможность зайти под логином Admin и получить форму для бана пользователей (по ajax)
(formadmin.html)
5) Для забаненных пользователей при следующей попытке отправки сообщения пропадает форма отправки
сообщения (по ajax) (formclient.html)
6) Имеется возможность выйти из чата.
7) Попытка перейти на страницу для ввода логина уже залогиненному пользователю не пройдет.
(ToMainPageServletFilter)
8) Используется СУБД H2.
9) В CleanerUsers листенере в предестрое контейнера очищаю список юзеров.
10) Встроен Jetty для быстрого запуска.
